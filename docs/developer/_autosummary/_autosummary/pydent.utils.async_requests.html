Executes an asynchronous function

Parameters

  • fxn (callable) – function to execute in parallel

  • arg_list (list of tuples) – list of arguments as tuples for each function run

  • kwargs (dict) – key-value arguments for each function run

  • chunk_size (int) – number of functions to run for each worker

  • desc (basestring) – description

  • progress_bar (boolean) – whether to display tqdm progress bar

Returns

list of results in same order as arg_list
