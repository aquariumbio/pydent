Trident (V0.0.17)

Trident is a python scripting API wrapper for Aquarium. With Trident, you can create python scripts and apps that mine data, create samples, submit laboratory protocols, and more. You are looking at documentation for version 0.0.17.

Installation¶

Requirements¶

  • python3.6
  • pip3

pipenv installation¶

It is recommended you install Trident using pipenv. Pipenv is now the officially recommended Python packaging tool from Python.org. It will install packages on a per project basis by creating a virtualenv for your package containing all of trident’s necessary dependencies.

pipenv trident installation¶

To install trident, download trident from github, move into trident’s root directory and run:

make

That’s it. You’ll be able to install pydent to your projects (see below)

pipenv trident environment¶

To open the environment containing all of trident’s dependencies, run

cd path/to/trident
pipenv shell

You can run any terminal command in trident’s environment using

pipenv run [command]

To run a python interpreter

cd path/to/trident
pipenv run python

..and from there you can import trident

import pydent

# stuff with trident/pydent

installing trident from remote¶

As soon as we create an official release, trident can be installed from remote via git using:

pipenv install git+https://github.com/klavinslab/trident.git@VER#egg=pydent
