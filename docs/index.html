Trident Documentation¶

Trident is the Python scripting API for Aquarium, the software that runs the UW BIOFAB. The code respository is freely available to members of the Klavins lab GitHub organization, which consists of lab members and collaborators. There is also a Javascript version of Trident available to that group.
